# 🌐 Hansika Varre — Portfolio Website

My personal portfolio website built with HTML, CSS and JavaScript, hosted on GitHub Pages.

🔗 **Live at:** [hansikavarre.github.io](https://hansikavarre.github.io)

---

## ✨ Features
- ✅ Responsive dark themed design
- ✅ Smooth scroll animations
- ✅ Projects showcase
- ✅ Skills section
- ✅ Certifications section
- ✅ Contact section with LinkedIn and Email

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

---

## 📌 Sections
- **Home** → Introduction and hero section
- **About** → Background, CGPA, achievements
- **Skills** → Languages, frameworks, domains
- **Projects** → All major projects
- **Certifications** → Completed certifications
- **Contact** → LinkedIn and Email links
